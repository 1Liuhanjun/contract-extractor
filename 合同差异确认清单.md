# 合同差异确认清单

> **说明**：基于《一级干线运输外包服务协议》（中国邮政 × 北京博华物流，安徽合肥），
> 逐一对比合同原文与手工填写的参考答案，标记所有差异点。
>
> 每项均列出：①合同原文证据 ②AI基于合同的提取结果 ③参考答案中的值

---

## 差异一：客户名称 — 缺了"局"字

| 维度 | 内容 |
|---|---|
| **合同原文** | 发包方（甲方）：**中国邮政集团有限公司安徽省合肥邮区中心** |
| | 合同从首页到尾页，所有出现该名称的位置均无"局"字 |
| **AI提取** | 中国邮政集团有限公司安徽省合肥邮区**中心** |
| **参考答案** | 中国邮政集团有限公司安徽省合肥邮区中心**局** |
| **差异** | 合同写"**中心**"，参考答案写"中心**局**"，多了一个"局"字 |

**需要确认**：甲方的正式工商注册全称究竟是什么？
- 如果正式全称是"**中国邮政集团有限公司安徽省合肥邮区中心局**"，说明合同文本漏写了"局"字，AI应该在提取时自动补全
- 如果正式全称就是"**中国邮政集团有限公司安徽省合肥邮区中心**"，那参考答案写错了
- 这条会影响所有与该客户的合同

---

## 差异二：账期 — 合同写30天，参考答案45天 ⚠️ 最重要

| 维度 | 内容 |
|---|---|
| **合同原文** | 第四条第（二）项第4款： |
| | *"甲方收到乙方开具的符合甲方要求的国家正式的**增值税专用发票之日起30日内**将相应邮路运费付至乙方指定账户内（如遇法定节假日及休息日顺延）。"* |
| **AI提取** | **30天** |
| **参考答案** | **45天** |
| **差异** | 合同明确写30天，参考答案填45天，差15天 |

**需要确认**：
- 如果实际执行账期就是45天——那这个信息从哪来的？是另有补充协议、招标文件规定、还是双方口头约定？
- 如果30天是对的——那参考答案这条填错了
- 这条影响财务付款，必须确认清楚

---

## 差异三：保证金 — 精度不同

| 维度 | 内容 |
|---|---|
| **合同原文** | 第十五条：*"乙方须向甲方缴纳保证金**3178420元**（按照邮路全年运费的10%收取）。"* |
| **AI提取** | **317.842万元**（3178420 ÷ 10000 = 317.842，精确计算） |
| **参考答案** | **317.8万元**（四舍五入保留一位小数） |
| **差异** | 精度不同，差0.042万元（420元） |

**建议**：统一规则即可。建议保留**一位小数**与参考答案一致，或保留两位小数更精确。

---

## 差异四：税率 — 合同没直接写9%

| 维度 | 内容 |
|---|---|
| **合同原文** | 甲乙方账户信息中：*"纳税人类型：**一般纳税人**"* |
| | 结算条款提到：*"开具正式**增值税专用发票**"* |
| | 但**没有出现"9%"**这个数字 |
| **AI推断** | **9%**（一般纳税人 + 货物运输服务 = 法定税率9%） |
| **参考答案** | **9.00%** |
| **差异** | 合同没直接写9%，但答案都是9%。AI需要做推断 |

**需要确认**：是否接受AI做这个推断？
- 接受 → AI根据"一般纳税人+运输服务"输出9%，格式统一为9.00%
- 不接受 → AI输出"一般纳税人（增值税专用发票）"，留空让业务填

## 差异五：日期格式

| 字段 | 合同原文 | AI提取 | 参考答案 |
|---|---|---|---|
| **合同开始时间** | 签字页两个日期：2022年7月29日、2022年9月29日 | 2022-09-29 | 2022/9/29 |
| **合同结束时间** | 自签订之日起2年，至下一轮采购完成最多不超过4个月 | 2024-09-28 | 2024/9/28 |

**说明**：
- 内容一致，仅格式不同
- 开始时间取最后一个签章日期（2022-09-29），逻辑无争议
- 结束时间的延长条款"最多不超过4个月"在参考答案中未体现（未延长到2025-01-28）

**建议**：统一为 YYYY-MM-DD 格式以对齐机器处理。

## 差异汇总

| # | 字段 | 差异类型 | 严重程度 | 处理方式 |
|---|---|---|---|---|
| 1 | **客户名称**（中心 vs 中心局） | 内容差异 | ⭐⭐⭐ | **需确认甲方正式全称** |
| 2 | **账期（天）**（30天 vs 45天） | 内容差异 | ⭐⭐⭐⭐⭐ | **必须确认**——合同写30天，人工填45天 |

| 6 | 补偿比例（10% vs 10.00%） | 格式差异 | ⭐ | 统一为两位小数 |

## 真正需要业务确认的只有2项

```
账期：合同写30天 → 参考答案45天 → ❓ 到底哪个对？
客户名称：合同写"中心" → 参考答案"中心局" → ❓ 正式全称到底是哪个？
```

其余4项（保证金精度、税率推断、日期格式、补偿比例格式）统一规则即可，不需要找人确认。
